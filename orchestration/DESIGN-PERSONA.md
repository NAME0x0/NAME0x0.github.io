# Unified Design Persona — Muhammad Afsah Mumtaz

Synthesized from ChatGPT brief, Gemini 3.1 Pro brief, and self-description.
Conflicts resolved by choosing the more opinionated or authentic take.

---

## The Core Tension (And Why It Matters)

Both AIs independently identified "The Sovereign Architect" — a cold,
clinical, brutalist systems builder. But the self-description reveals
something they both underweighted:

> "creative, forward-thinking, **rebellious**, experimental, efficient,
> **lazy**, and gifted"

> Music: **Pop, R&B, Lo-fi, Classical**

> Feeling I want: **A sense of awe and wonder**

This is not a cold engineer. This is a **gifted rebel who builds cathedrals
and walks away before the paint dries**. The portfolio must capture BOTH:
the sovereign systems depth AND the effortless creative audacity. It should
feel like someone built something impossibly beautiful and didn't even break
a sweat.

**Design implication:** The site should NOT feel labored, over-engineered,
or clinical. It should feel like raw talent channeled through precise
systems. Brutalism with soul. A void that breathes.

- **Trackmania mindset** — endless room for improvement. The portfolio is never

## Aesthetic Definition

### The Feel

**Not this:** Cold laboratory. Clinical facility. Brutalist concrete.
**Not this either:** Playful, colorful, casual, "creative agency."
**This:** A void that contains something sacred. Like standing in a dark
cathedral and seeing the light come through a single window. Quiet power.
The kind of room where you whisper, not because you have to, but because
the space commands respect.

### The Reference Constellation

| Dimension | Primary Reference | Why |
|-----------|-------------------|-----|
| **Architecture** | Tadao Ando (concrete + light + void) | Brutalist structure with spiritual atmosphere |
| **Film** | *Ghost in the Shell* (1995) + *Arrival* | Systems depth + alien beauty + reverence |
| **Game** | Trackmania (endless improvement) | The pursuit itself IS the aesthetic |
| **Music** | Lo-fi meets Classical | Calm creative flow with moments of grandeur |
| **Art** | Constructivism + Generative Art | Structural utility + organic unpredictability |
| **Fashion** | Monochrome techwear | Functional, minimal, all-black, no logos |
| **Design** | Swiss Typography + Dieter Rams | Grid discipline + "less but better" |

### The Palette (Confirmed)

| Token | Hex | Emotional Role |
|-------|-----|----------------|
| `void` | `#000000` | The infinite. Not "dark mode" — the void itself. |
| `ink` | `#E8E4DE` | The human mark. Warm, analog, imperfect white. |
| `ink-dim` | `#8A8578` | The whisper. Secondary information. |
| `ink-faint` | `#3A3832` | The structure. Visible only if you look for it. |
| `accent` | `#C4B5A0` | The warmth. A hint that a human built this. |

**Why off-white, not pure white:** Pure white (#FFFFFF) is clinical. Off-white
(#E8E4DE) is warm, like aged paper or bone. It implies history, craft, and
humanity. This is critical — the warmth in the palette is what separates this
from every other "dark portfolio."

### Typography (Refined)

| Role | Font | Why |
|------|------|-----|
| **Headings** | Space Grotesk | Geometric but not cold. Has personality. |
| **Body** | Manrope | Clean, readable, slightly warm. |
| **System/Code** | Geist Mono | Modern monospace. Ties to the terminal identity. |

**Gemini suggested Inter/Helvetica** — rejected. Too generic, too corporate.
Space Grotesk has the right balance of geometric precision and character.

## Project Presentation

### Gemini's "System Readouts" (Adopted, Refined)

Projects should NOT be presented as portfolio case studies (too corporate)
or as a simple card grid (too basic). They should feel like accessing
system documentation.

Each project card contains:
- **Name** — the system identifier (MALD, MAVIS, AVA, etc.)
- **One-line description** — what it does, concisely
- **Stack tags** — the technologies, in monospace
- **Layer badge** — which Sovereign Stack layer it belongs to
- **Link** — GitHub arrow, nothing else

On hover or expansion (future): show the "readout" —
- **Problem** → what gap existed
- **Architecture** → how it's built
- **Status** → active / stable / experimental

### Proud Projects (from self-description)

The user specifically called out **SMNTC**, **OMNI**, and **AVA** as their
proudest work. These should be given visual priority:
- Positioned first in the grid (regardless of filter)
- Potentially a slightly different card treatment (featured badge or expanded)

**Note:** OMNI was not in the original curated project list. Need to ask the
user about this project and add it to `lib/data/curated.ts`.

## Anti-Patterns (Unified, Final)

### NEVER Do These

| Anti-Pattern | Why |
|-------------|-----|
| Neon pink/cyan color schemes | Crypto-bro aesthetic, not sovereign |
| Bouncy/elastic animations | Playful, not authoritative |
| "Hi, I'm passionate" copy | Generic, beneath this identity |
| Skill bar percentages | Arbitrary, meaningless, overused |
| Logo wallpaper for tech stack | Cluttered, says nothing |
| Rainbow gradients | Web3 cliche, not this person |
| Floating 3D coins/objects | Gimmicky, distracting |
| Cluttered dashboards | Contradicts minimalism |
| Light mode toggle | This identity IS the void. No compromise. |
| Social proof counters | "X visitors" or "Y stars" prominently = insecure |
| Template-looking layouts | The user said all websites are "pretty basic" — this must be unprecedented |

### BE CAREFUL With These

| Element | Risk | Mitigation |
|---------|------|------------|
| Glassmorphism | Can look generic/Apple-y | Use smoked obsidian, not frosted white |
| Terminal aesthetic | Can feel like a novelty theme | Keep it subtle — overlines, not full terminal UI |
| Grid/matrix patterns | Can look like a screensaver | Keep opacity < 10%, decorative only |
| Scroll animations | Can feel like a marketing site | Keep them understated — fades and slides only |
