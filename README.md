## 🎭 **Features**

### **Cinematic Sections**
- 🌟 **Prologue**: Particle genesis animation - dot morphs into glowing nebula, then interactive sphere
- 🌍 **Origins**: Interactive 3D globe tracing life journey across countries
- 💪 **Superpowers**: 3D rotating skill rings that dissolve into radar chart
- 🧪 **Lab Experiments**: Parallax masonry with flip cards showing projects
- 🤖 **AI Command Center**: Fully functional terminal with 15+ commands
- 📱 **Daily Ops**: Glass morphism carousel with real-time widgets
- 📧 **Signal Booster**: Contact form with typewriter animations

### **Technical Highlights**
- ⚡ **Blazing Performance**: < 250KB total bundle, TTI < 900ms
- 🎬 **60fps Animations**: GSAP ScrollTrigger with smooth scroll effects
- 🌌 **3D Graphics**: Three.js particle systems and interactive models
- 📊 **Data Visualization**: Chart.js radar charts and interactive elements
- 🎨 **Glass Morphism**: Modern UI with backdrop-filter effects
- 🌓 **Theme Toggle**: Smooth dark/light mode transitions
- 📱 **PWA Ready**: Installable with offline support
- ♿ **Accessible**: WCAG 2.1 AA compliant with screen reader support

## 🚀 **Quick Start**

### **Prerequisites**
- Node.js 18+ 
- npm 9+

### **Installation**
```bash
# Clone the repository
git clone https://github.com/NAME0x0/futuristic-portfolio-prototype.git
cd futuristic-portfolio-prototype

# Install dependencies
npm install

# Start development server
npm run dev
```

### **Available Scripts**
```bash
npm run dev      # Start development server (localhost:5173)
npm run build    # Build for production
npm run preview  # Preview production build
```

## 🛠 **Tech Stack**

### **Core Technologies**
- **Build Tool**: Vite 6.x for lightning-fast development
- **Styling**: CSS3 with design tokens and glass morphism
- **Animations**: GSAP ScrollTrigger for cinematic effects
- **3D Graphics**: Three.js for particles, globe, and skill rings
- **Charts**: Chart.js for radar visualization
- **PWA**: Service Worker with manifest for app-like experience

### **Architecture**
- **Modular JavaScript**: ES6+ modules with dynamic imports
- **Component-Driven**: Reusable patterns and design tokens
- **Performance-First**: Code splitting and lazy loading
- **Accessibility-First**: Semantic HTML and ARIA labels

## 🎨 **Design System**

### **Colors**
- **Primary**: `#FF5C00` (Futuristic Orange)
- **Background**: `#000000` (Dark) / `#F5F5F5` (Light)
- **Text**: `#FFFFFF` (Dark Mode) / `#000000` (Light Mode)
- **Glass**: `rgba(255, 255, 255, 0.1)` with blur effects

### **Typography**
- **System Fonts**: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto
- **Fluid Scaling**: clamp() functions for responsive text
- **Hierarchy**: 1.5× scale ratio for headings

### **Animations**
- **Easing**: Custom cubic-bezier curves for natural motion
- **Duration**: 0.3s for micro-interactions, 1-2s for scene transitions
- **Respect**: `prefers-reduced-motion` for accessibility
